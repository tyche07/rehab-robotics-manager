# Rehab Robotics Manager

Smart rehabilitation robotics management platform built with Firebase.

## Features
- Smart scheduling
- Protocol-based rehab
- AI-assisted therapy
- Compliance & audit logs

## Tech Stack
- Firebase
- React / Next.js
- Firestore
- Cloud Functions

## Setup
1. Clone repo
2. Add `.env` file
3. Run `npm install`
4. Run `npm run dev`
