# **App Name**: Rehab Robotics Manager

## Core Features:

- Patient Profile Management: Create and manage patient profiles, including medical history and therapy goals.
- Session Control: Allow real-time monitoring of biometric data and basic control of the therapy robot during sessions.
- Performance Analysis: Provide detailed charts and graphs visualizing patient progress.
- AI Powered Adjustment: Using sensor input from the robotics platform as a tool, determine in real-time whether or not to adjust therapy parameters dynamically based on patient progress and safety metrics.
- Report Generation: Automatically generate detailed reports of therapy sessions for each patient.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to create a calming and clinical atmosphere.
- Background color: Off-white (#F0F8FF) to maintain a clean, professional look.
- Accent color: Gentle green (#B6F498) to highlight important data and interactive elements.
- Body and headline font: 'PT Sans' sans-serif for a modern, clean aesthetic.
- Use clear, professional icons for therapy controls and data visualization.
- Ensure a clear, intuitive layout with easy navigation to key features.
- Implement subtle transitions for smooth data updates and feedback.
